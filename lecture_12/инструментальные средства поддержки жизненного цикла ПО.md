# Инструментальные средства поддержки жизненного цикла ПО

- инструментальные средства - то, что помогает сделать разработку ПО более эффективным

## Управление жизненным циклом ПО

- SDLC - Sortware Development Lifecycle Managment
- ALM - Application Lifecycle Management

- Жизненный цикл
    - разработка ПО
    - Эксплуатация ПО

- Составляющие жизненного цикла ПО

    - управление требованиями 
    - проектирование ПО
    - разработка ПО
    - управление качество ПО
    - развертывание ПО
    - обслуживание и поддержка

### Инструментальные средства
-  разработка и отладка ПО
- Управление артефактами

- управление проектами

- дополнительные

### Инструментальные средства разработки и отладки ПО

- интегрированные среды разработки (IDE)
- интеллектульные текстовые редакторы
- отладчики
- профилировщики (цель - профильная обработка приложения? valgrind)
- системы аудита ПО (code review, code inspections)

- системы автоматизации сборки

- системы непрерывной интеграции, непрерывного развертывания и непрерывной доставки (CI/CD)(jenkins, teamCity)

- системы хостинга проекта (gitHub, GitLab..)
(хранение проектов)

- порталы проектов
